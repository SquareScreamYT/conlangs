# Pwilsekuun

Pwilsekuun is the language spoken in Yellowgrass.

## Phonology

### Consonants

|                        | Labial    | Alveolar  | Palatal | Velar |
|------------------------|-----------|-----------|---------|-------|
| Nasal                  | m mʲ      | n         | ɲ~nʲ    | ŋ ŋʲ  |
| Voiceless Stop         | p pʲ      | t         | c~tʲ    | k kʲ  |
| Voiced Stop            | b bʲ      | d         | ɟ~dʲ    | g gʲ  |
| Affricate              |           | t͡s t͡sʲ    | t͡ɕ      |       |
| Non-Sibilant Fricative | f v fʲ vʲ |           | ç~hʲ    | h     |
| Sibilant Fricative     |           | s z sʲ zʲ | ɕ ʑ     |       |
| Lateral                |           | l         | ʎ~lʲ    |       |
| Trill                  |           | r rʲ      |         |       |
| Approximant            | w         | ð̞ ð̞ʲ      | j       | ɥ~wʲ  |

* Palatalisation is written as [j]
* [t͡s] is written as ⟨c⟩
* [ɲ c ɟ t͡ɕ ɕ ʑ] are written as ⟨nj kj gj cz sz zs⟩
* [ð̞] is written as ⟨dr⟩

### Vowels

|       | Front | Back |
|-------|-------|------|
| Close | i y   | u    |
| Mid   | e ø   | o    |
| Open  | æ     | a    |

* [ø æ] are written as ⟨ö ä⟩

### Phonotactics

(C)(C)V(C)(C), where

* C is any consonant
* V is any vowel

## Syntax

### Cases

Pwilsekuun has 6 cases and differentiates between singular and plural.

|     | SG  | PL  |
|-----|-----|-----|
| NOM | -n  | -f  |
| ACC | -k  | -nj |
| GEN | -zs | -z  |
| DAT | -dr | -s  |
| LOC | -h  | -k  |
| INS | -l  | -sz |

Example:

```text
hjöson
onion.SG.NOM
"the onion"
```

### Pronouns

Pronouns are formed by adding the person and the case together.

| 1    | 2   | 3   |
|------|-----|-----|
| cze- | ci- | sä- |

Example:

```text
säzs    lazson
3SG.GEN bird.NOM
"their bird"
```

### Tense

| PST | PRS | FUT |
|-----|-----|-----|
| -m  | -b  | -v  |

Example:

```text
sän     releb
3SG.NOM eat.PRS
"they eat"
```

Pwilsekuun uses SVO word order.

## Introduction

This guide describes Gaselian, a language originally spoken around the Chielbaric Peninsula.

Gaselian is a polysynthetic, head-final language in which words frequently encode information that would correspond to entire sentences in English. Grammatical relationships are primarily expressed through suffixes, and roots typically appear in bound form. The language is phonotactically simple, with extensive morphophonemic alternations across morpheme boundaries, similaer to Greenlandic (Kalaallisut).

Unless otherwise noted, phonetic transcriptions are given in IPA. The orthography will usually be presented alongside the phonetic transcription.

## Phonology

The phonemes listed are all in the International Phonetic Alphabet. All phonemes are written as in the IPA, except when surrounded by angle brackets `<>`.

### Consonants

Gaselian has 12 consonants:

|           | Labial | Alveolar | Palatal | Dorsal |
|-----------|--------|----------|---------|--------|
| Nasal     | m      | n        |         |        |
| Plosive   | b      | d        | ɟ \<j>  | g      |
| Fricative | v      | z        |         | ɦ \<h> |

### Vowels

Gaselian has 4 vowels:

|       | Front | Central | Back |
|-------|-------|---------|------|
| Close | i     |         | u    |
| Mid   | e     | ə \<y>  |      |

### Diphthongs

The following diphthongs are valid:

|   | a  | i  | u  | e  | y  |
|---|----|----|----|----|----|
| a |    | ai | au |    |    |
| i | ia |    | iu | ie | iy |
| u | ua | ui |    | ue | uy |
| e | ea | ei | eu |    | ey |
| y |    | yi | yu |    |    |

### Allophony

Voiced obstruents are neutralized in word-final position. In this environment, /b d ɟ g v z ɦ/ surface as their voiceless counterparts [p t c k f s h], respectively.

### Phonotactics

(C)V(V)(C) where:

* C is the consonant
* V is vowel or diphthong

### Orthography

Vowels in hiatus are written with a `i`:

* [a.a] `a'a`
* [i.u] `i'u`
* [ə.i] `y'i`

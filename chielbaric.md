# Chielbaric

## Phonology

### Consonants

|           | Bilabial   | Dental   | Alveolar     | Postalveolar | Palatal | Dorsal     | Glottal  |
|-----------|------------|----------|--------------|--------------|---------|------------|----------|
| Nasal     | m \<м>     |          | n \<н>       |              | ɲ \<њ>  | ŋ \<ҥ>     |          |
| Plosive   | p b \<п б> |          | t d \<т д>   |              |         | k g \<к ґ> | ʔ \<ъ>   |
| Fricative | f v \<ф в> |          | s z \<с з>   | ʃ ʒ \<ш ж>   | ç \<ь>  | x ɣ \<х ғ> | h~ɦ \<г> |
| Affricate |            |          | t͡s d͡z \<ц ѕ> | t͡ʃ d͡ʒ \<ч џ> |         |            |          |
| Trill     |            |          | r \<р>       |              |         | ʀ~ʁ \<ӷ>   |          |
| Liquid    | w \<ў>     | ð̞~ð \<ҙ> | l \<л>       | ɥ ɨ̯~ɹ \<ю̆ ы̆> | j \<й>  |            |          |
| Lateral   |            |          | ɬ \<ԯ>       |              | ʎ \<љ>  |            |          |

### Vowels

|       | Front      | Central | Back   |
|-------|------------|---------|--------|
| Close | i y \<и ю> | ɨ \<ы>  | u \<у> |
| Mid   | e ø \<е ё> | ə \<э>  | o \<о> |
| Open  | æ~ɛ \<я>   | a \<а>  |        |

Some vowels can be lengthened. It is marked with an acute `´`.

|       | Front        | Central | Back    |
|-------|--------------|---------|---------|
| Close | iː yː \<и́ ю́> |         | uː \<у́> |
| Mid   | eː øː \<е́ э́> |         | oː \<о́> |
| Open  | æː~ɛː \<я́>   | aː \<а́> |         |

## Phonotactics

(C)(C)V(C)(C) where:

* C is any consonant
* V is any vowel

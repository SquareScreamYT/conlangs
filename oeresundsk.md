## Phonology

### Consonants

|             | Labial | Coronal | Palatal | Dorsal  |
|-------------|--------|---------|---------|---------|
| Nasal       | m      | n       |         | ŋ \<ng> |
| Stop        | p b    | t d     |         | k g     |
| Fricative   | f v    | s       | ɕ \<sj> | x \<h>  |
| Approximant |        | l       | j       |         |
| Trill       |        | r       |         |         |

* /x/ can be realised as [h]
* /ɕ/ can be realised as [ʃ] or [ɧ] or [ʂ]
* /r/ can be realised as [ɾ]
* /v/ can be realised as [w] or [ʋ]

### Vowels

|           | Front  | Back   |
|-----------|--------|--------|
| Close     | i y    | u      |
| Close-Mid | e ø    | o      |
| Open-Mid  | ɛ \<æ> | ɔ \<å> |
| Open      | a      |        |

* Stress always falls on the second to last syllable.
